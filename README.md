# 거상 캐시 거래 장부

거상 캐시와 지전 거래를 정리하고 계산하는 웹 기반 계산기입니다.

## 배포 주소

[https://dev-gom.github.io/gersang_utils/](https://dev-gom.github.io/gersang_utils/)

## 기술 스택

- React 19
- Vite 6
- GitHub Pages

## 주요 기능

### 1. 지전으로 캐시 구매

게임머니(억) 단가와 시세 비율을 입력하면 캐시 가격을 자동 계산합니다.

### 2. 캐시로 지전 구매

캐시 단가와 시세 비율을 입력하면 게임머니 가격을 자동 계산합니다.

### 3. 프리셋 관리

자주 입력하는 아이템 조건을 저장한 "프리셋"으로 관리할 수 있습니다.

- 현재 입력 값을 프리셋으로 저장
- 프리셋 불러오기: 입력란에 자동 채우기
- 프리셋 추가: 바로 장부에 반영
- 프리셋 삭제

### 4. 장부 관리

현재 거래 장부를 이름 붙여 저장하고, 나중에 다시 불러올 수 있습니다.

### 5. 파일 저장/불러오기

모든 데이터(프리셋, 장부)를 JSON 파일로 백업하거나 불러올 수 있습니다.

- 파일로 저장: 모든 데이터를 `.json` 파일로 다운로드
- 파일 불러오기: 백업 파일을 불러와 기존 데이터에 추가 또는 덮어쓰기
